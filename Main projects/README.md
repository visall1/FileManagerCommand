
## Compilation

To compile the project, you need a C++17 compatible compiler. The following instructions use `g++` and `clang++`, but you can use any compiler that supports C++17.

### Using `g++`:

```sh
g++ -std=c++17 -o file_manager main.cpp FileManager.cpp CommandHandler.cpp


```Run 
./file_manager or file_manager.exe
